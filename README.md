# cargoAllCleanGo

Recursively finds and cleans Rust projects (containing `Cargo.toml`) starting from a specified directory.

## Usage

Run the executable, optionally providing a root directory path. If no path is provided, it defaults to the current working directory.

```bash
cac /path/to/projects
```

## Build

```bash
git clone https://github.com/Uliboooo/cargo_all_clean
cargo build --release
```
